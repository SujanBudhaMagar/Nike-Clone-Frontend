# 👟 Nike Frontend Clone

> A pixel-perfect, modern **Nike website frontend clone** built with **Next.js, React, and Tailwind CSS** ⚡  
> This project is just for **learning, practice, and portfolio showcase** (no commercial use).

---

## ✨ Highlights

✅ **Hero Banner & Carousel** — Sleek product sliders with Swiper  
✅ **Reusable Components** — Cards, Navbars, Footers, Buttons  
✅ **Fully Responsive** — Looks good on Desktop 💻, Tablet 📱, and Mobile 📲  
✅ **Clean Architecture** — Organized with constants, types, and components  
✅ **Dark Overlay & Modals** — Location selector and popups like the real site

---

## 🛠️ Tech Stack

- ⚛️ **Next.js 13+** – Fast React framework with App Router
- 🎨 **Tailwind CSS** – Utility-first styling
- 🌀 **Swiper.js** – Carousels for hero & product galleries
- 🛠 **TypeScript** – For strong typing and maintainability
- 🔗 **React Icons** – For lightweight, beautiful icons

---

## 📂 Folder Structure

├── public/ # Static assets (images, icons, fonts)
├── src/
│ ├── app/ # Pages (Next.js App Router)
│ ├── components/ # Reusable UI components
│ ├── constants/ # Static JSON-like data (menus, footer, products)
│ ├── styles/ # Global Tailwind setup
│ └── types/ # TypeScript interfaces/types
└── README.md

## 🚀 Getting Started

Clone this repo & start coding right away:

```bash
# 1️⃣ Clone the repo
git clone https://github.com/your-username/nike-frontend-clone.git
cd nike-frontend-clone

# 2️⃣ Install dependencies
npm install   # or yarn install

# 3️⃣ Start dev server
npm run dev
```

Now open 👉 http://localhost:3000
